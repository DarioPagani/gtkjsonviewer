GtkJsonView : A simple json viewer written in GTK.
==================================================

# Usage

  python gtkjsonview.py test.json

# Screenshot

  ![GtkJsonView](http://farm4.static.flickr.com/3529/3252639468_9c41d0e97f_o_d.png)

# Require

* Python 3
* Gtk module
* simplejson module

 sudo pip install simplejson
